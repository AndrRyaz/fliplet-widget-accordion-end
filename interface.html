End of Accordion help

How to use

This is one of a 2-part component:

  • Accordion heading
  • End of Accordion (this component)

An accordion group is a group of collapsable accordions, each collapsable accordion is created by an Accordion heading component, and you finish the accordion group by adding the End of Accordion component at the end of all the Accordion heading components. Using this method you can create several accordion groups with one or more collapsable accordions.

For more information, read our Accordion documentation.
